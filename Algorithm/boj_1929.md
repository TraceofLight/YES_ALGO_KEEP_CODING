## BOJ 1929. 소수 구하기

### 문제 링크

[[BOJ 1929] 소수 구하기](https://www.acmicpc.net/problem/1929)

### 분류

수학, 정수론, 소수 판정, 에라토스테네스의 체

### 코드
```python
def isPrime(num):
    if num==1:
        return False
    else:
        for i in range(2, int(num**0.5)+1):
            if num%i == 0:
                return False
        return True

M, N = map(int, input().split())

for i in range(M, N+1):
    if isPrime(i):
        print(i)
```
