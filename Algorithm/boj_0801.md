## 단어 뒤집기

```python
n = int(input())

for i in range(n):
    res = []
    a_dict = list(map(str, input().split()))
    for j in a_dict:
        result = j[::-1]
        res.append(result)
        
    b = ' '.join(res)
    print(b, end=' ')
        
```

- 공백을 기준으로 나누는 리스트를 생성한다

- 리스트 인자를 하나씩 받아서 슬라이싱으로 역순으로 바꾼다

- join을 이용해서 역순이 된 인자를 합쳐준다.



## 스택

```python
import sys
n = int(sys.stdin.readline())

stack = []
for i in range(n):
    command = sys.stdin.readline().split()

    if command[0] == 'push':
        stack.append(command[1])
    elif command[0] == 'pop':
        if len(stack) == 0:
            print(-1)
        else:
            print(stack.pop())
    elif command[0] == 'size':
        print(len(stack))
    elif command[0] == 'empty':
        if len(stack) == 0:
            print(1)
        else:
            print(0)
    elif command[0] == 'top':
        if len(stack) == 0:
            print(-1)
        else:
            print(stack[-1])
```

- 함수가 아닌 문자열 인식으로 프린트를 하는 방식



## 괄호

```python
T = int(input())
import sys
for _ in range(T):
    chars = str(sys.stdin.readline().split())
    stack = []
    opening = '('
    closing = ')'

    for i in chars:
        if i == opening:
            stack.append(i)
        elif i == closing:
            if len(stack) != 0 and stack[-1] == opening:
                stack.pop()
            else:
                stack.append(i)      
    if len(stack) == 0:
        print('YES')
    else:
        print('NO')

```
